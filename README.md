## Projects

### [Utility Bill Automation System](https://github.com/bhanu-devv/utility-bill-automation-system)
A Python desktop application that extracts meter-level data from utility bill PDFs, validates billing records, and updates Excel workbook templates — preserving existing formulas and reporting logic. Built with a Tkinter GUI so non-technical users can run it without code. Packaged as a standalone Windows `.exe` using PyInstaller.

**Tools:** Python · pdfplumber · openpyxl · Tkinter · Regular Expressions · PyInstaller

---

### [CSU EnergyStar Automation](https://github.com/bhanu-devv/csu-energystar-automation)
A Python ETL pipeline that cleans, validates, and prepares utility consumption records for upload to ENERGY STAR Portfolio Manager. Handles meter-to-building mapping, billing period validation, duplicate detection, and upload-ready row generation across multiple utility types (steam, electric, water, natural gas, chilled water).

**Tools:** Python · pandas · openpyxl · requests · python-dotenv · ENERGY STAR Portfolio Manager API

---

### [CSU Utilities Data Management System](https://github.com/bhanu-devv/csu-utilities-data-system)
A data management system concept for tracking and analyzing campus utility usage across buildings. The strongest implemented part is the enterprise database design, including a normalized SQL schema, stored procedures, triggers, indexing, and reporting queries developed for IST 634 Enterprise Databases at CSU. The repo also documents planned backend and dashboard extensions.

**Tools:** SQL · PostgreSQL/MySQL concepts · Database Design · Stored Procedures · Triggers · Indexing · Excel
